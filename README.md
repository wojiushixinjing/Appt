# Appt
紧紧围绕医务工作者职业成长路径展开，现阶段通过高质量的数字医疗技术手段，以陪诊服务切入点，智慧连接医务工作者，致力于创建一个真实的医护社区，为医护提供人脉拓展、陪诊服务、诊前预约、预约转运车、院内代约检查、代办去送报告、病理切片、复印病历、代办买药、预约代办出入院、产检服务权益卡及患者管理等一系列功能。
